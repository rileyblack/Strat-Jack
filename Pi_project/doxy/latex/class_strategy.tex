\doxysubsection{Detailed Description}
Class which determines the approprate strategyon agiven turn during a blakjack game. 

\mbox{\hyperlink{class_strategy}{Strategy}} object contains two matrices where each index is a blackjack move, the sum of the \mbox{\hyperlink{class_user}{User}} and \mbox{\hyperlink{class_dealer}{Dealer}} hands indicate the approriate x and y index respectively the appropriate strategy grid is used, depending on if the hand type is a hard total or soft total
