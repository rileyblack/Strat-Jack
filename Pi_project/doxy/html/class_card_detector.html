Detailed Description

class for recognizing the ranks of captured playing cards

obtains images encoded into Mat objects, and then finds and extracts the top left corner of the captured image and compares it to local images of playing cards to determine the strongest match and if the match is sufficiently strong Code Heavily Influenced from github.com/arslanxone/playingCardsDetection/blob/master/BVPoker.cpp
